7月9日Linux命令的使用

1.命令结构

命令 -选项 参数

注意事项

  • 多个选项可以写在一起
  • 多个选项不区分选项的顺序

2.系统级命令

命令 命令全称 命令含义
mkdir make directory 创建目录
ls list 显示目录中的文件
touch touch 创建文件
rm remove 删除文件
cd change directory 切换目录

3.yum命令

命令 命令含义
yum list 显示软件列表
yum install APPNAME 安装指定软件
yum search APPNAME 搜索相关软件
yum check-update 检查软件是否需要更新
yum update APPNAME 更新指定软件
yum update 更新所有软件
yum remove APPNAME 卸载指定软件
